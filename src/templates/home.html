Named Entity Recognition using LSTM with Keras

Named Entity Recognition (NER) is a Natural Language Processing (NLP) task that involves identifying and classifying entities extracted from text into predefined (named) categories such as person, location, organization, etc. Deep learning methods such as Long Short-Term Memory (LSTM) networks have been used for NER due to thier ability to capture sequences of information and store thier context.

For this project, the selected deep learning library is Keras. Keras provides accessible APIs for building and training deep learning models. The main objective of this project is to develop a model that automatically classifies named entities from text. This has many applications including quick information extraction, machine translation, and sentiment analysis.

Some examples of NER use cases include:

  • Social media analysis to extract user's names, locations, and products mentioned in tweets.

  • News articles analysis to extract people, organizations, and locations mentioned in the article for better indexing and searchability.

  • Medical text analysis to extract medical terms, drug names, and dosage instructions for automated diagnosis and treatment suggestions.

  • Customer service analysis to extract customer names, product names, and customer feedback for better customer experience.

The Annotated Corpus for Named Entity Recognition dataset was used to build this model.

Python code used to build the model: Jupyter Notebook and Python Script
